Detailed Description

Creates image objects to render to web canvas.

Usage example:

SDL_Surface* target;
Image* bg;
// Load PNG assets
bg = new Image("assets/bg.png");
if (!bg) {
std::cerr << "Failed to load background image." << std::endl;
}
// Draw background image
bg->Stage(target, 0, 0);
