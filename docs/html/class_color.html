Detailed Description

Creates RGBA color object.

Usage example:

void useColorClass(const Color& color)
{
// Do something with the color
uint8_t red = color->r;
}
useColorClass(Color::transparent());
useColorClass(Color::Color(255, 128, 0, 0));
